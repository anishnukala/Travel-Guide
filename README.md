# 🌍 Travel Guide

## 📌 Overview

This project is a travel-themed product catalog website built with HTML, CSS, and Bootstrap. The site helps users explore student-friendly travel ideas and useful travel products in a clean, responsive interface.

The website contains three main pages:
- 🌆 A destinations page with budget-friendly travel locations  
- 🎒 An essentials page with useful travel products  
- ❓ An FAQ page with common travel questions  

The goal of this project is to demonstrate structured web design, responsive layouts, navigation between pages, and organized presentation of content.

## 📄 Pages

- `TravelDestinations.html` — 🌆 travel destinations with images, descriptions, and estimated weekly budgets  
- `TravelEssentials.html` — 🎒 travel products with images, descriptions, and prices  
- `faq.html` — ❓ frequently asked questions using an accordion layout  

## ✨ Features

- 📱 Responsive Bootstrap navbar for navigation between pages  
- 🗺️ Destination cards with descriptions and budget estimates  
- 🧳 Travel essentials cards with product details and prices  
- 📂 FAQ page using a Bootstrap accordion  
- 🎨 Shared footer and consistent styling via `assets/styles.css`
